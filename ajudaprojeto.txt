1. Modelagem com Structs
Para os atores (que são vários), o ideal é usar uma matriz de caracteres ou uma string longa. Para as salas e sessões, usamos tipos básicos.

typedef struct {
    int codigo;
    char nome[50];
    int capacidade;
    char tipoExibicao[20]; // 2D, 3D, IMAX
    int acessivel;         // 1 para sim, 0 para não

3. A Lógica das Chaves Únicas (Validação)
Antes de cada incluir, você precisa chamar a função de buscar.
Se buscar retornar um índice (0, 1, 2...), significa que a chave já existe -> Erro.
Se buscar retornar -1, significa que o código está livre -> Pode incluir.
4. Por que usar int no retorno em vez de void?
Para as funções de Inclusão, Alteração e Exclusão, recomendo retornar um int para sinalizar o que aconteceu:
int incluirSala(BancoDados *db):
Pede o código.
Busca no db->vSalas.
Se existir, return 0 (Erro: Chave duplicada).
Se não existir, usa realloc para aumentar o vetor, salva e return 1 (Sucesso).

void salvar_salas(){
    FILE *f = fopen("salas.dat", "wb");
    fwrite(vSalas, sizeof(Sala), qtdSalas, f);
    fclose(f);
}

void carregar_salas(){
    FILE *f = fopen("salas.dat", "rb");
    if(!f) return;

    fseek(f, 0, SEEK_END);
    qtdSalas = ftell(f) / sizeof(Sala);
    rewind(f);

    vSalas = malloc(qtdSalas * sizeof(Sala));
    fread(vSalas, sizeof(Sala), qtdSalas, f);

    fclose(f);
}
